## なぜアーカイブするの？
このサイトでは、/から始まるディレクトリパス、通称「絶対パス」や、リンク先サイトの仕様変更により、リンクが切れたものが多いです。
そのため、このまま使うのは異常です。
また、サイトが近代仕様に整っていないことや、CSSの複雑・かつ無駄なコードの量の多さから逃れる最適解は何かと模索した結果、このサイトを**アーカイブ**することによって、問題を解決します。
